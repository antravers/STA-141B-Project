Web Scrapping¶

The Costco Price List table, last updated the 16th of February 2023, from the website, https://www.simplegrocerydeals.com/costco-price-list/ was used to get price and product data for Costco. By the methods of Pythons Requests module and BeautifulSoup the table of interest was discovered to be "dynamic content", however upon inspecting the html of the website a link attached to the table (https://supergrocerylists.com/colists/list/418/lookup) was found displaying just the table of interest. The table was retrieved by using Pandas read html function to scrape the table.

The raw table had 169 rows and 5 columns, the variables contained in the table are Product, Brand, Price, Size and Price per Unit. The list of 169 food products was varied in kind, from food products unique to Costco to general dairy and produce items.

Gathering nutritional data from Costco took longer than predicted since problems were found while following the plans prepared beforehand. Using the same method to collect the price list for the items' nutritional value proved problematic without additional cost. And Extracting the data automatically from Edamame API proved to also be challenging. Web scrapping using Edamame API to manually enter nutritional data in a spreadsheet was the best way to solve the problem. After manually finding the data, it is transferred and classified into total fat, trans fat, saturated fat, cholesterol, sodium, total carbs, dietary fiber, sugar, vitamin D, Calcium, Iron, and Potassium. Data that are not available are then considered to be close to 0 or irrelevant. After data collection, a final check-up was done to make sure each data was measured for a single piece so that it would ease the next steps needed for this project.

Optimization¶

To find the Stigler Diet Solution for Costco's, the same method used to solve the Stigler Diet problem for Safeway and Trader Joe's will be followed; the Primal-Dual Path-Following Method from SciPy library. The minimum constraints are set to be the FDA daily nutritional requirements concerning the nutrients present in Costco's products.
